# Payroll Management System

Day 1 frontend setup for a Payroll Management System internship project.

## Tech Stack

- React with Vite
- React Router DOM
- Tailwind CSS
- Supabase client

## Scripts

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file using `.env.example` as the template, then add the Supabase credentials when they are available.
